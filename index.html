Lab Guide

📋 How to Use

Drag chemicals from the list into the beakers. Combine them to see reactions!

🧪 Acid-Base Reactions

🧪+⚗️
HCl + NaOH → Neutralization (salt water)
☣️+⚗️
H₂SO₄ + NaOH → Neutralization
🧪+🧪
HCl + Na₂CO₃ → CO₂ bubbles!
☣️+🧪
H₂SO₄ + Na₂CO₃ → CO₂ bubbles!

🔵 Precipitation Reactions

🔵+⚗️
CuSO₄ + NaOH → Blue precipitate
🟤+⚗️
FeCl₃ + NaOH → Brown precipitate
✨+🧪
AgNO₃ + NaCl → White precipitate
✨+🧪
AgNO₃ + KI → Yellow precipitate
🟣+🧪
KMnO₄ + NaOH → Green then brown

⚡ Metal Reactions

🔥+🧪
Mg + HCl → Hydrogen bubbles!
⚡+🧪
Zn + HCl → Hydrogen bubbles
📦+🧪
Al + HCl → Slow hydrogen
⚖️+🧪
Pb + HCl → Very slow reaction
