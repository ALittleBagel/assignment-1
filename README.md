Website: Chem Lab

How to use:
1. Open the "Chem Lab.html" file in a web browser.
2. The main menu allows you to navigate to three sections:
    - Periodic Table: An interactive periodic table of elements. Click on an element to see its details.
    - Virtual Lab: A virtual chemistry lab where you can mix chemicals and see the reactions.
    - Quiz: A quiz to test your chemistry knowledge.
